Validador de Expresiones

Verifica si tus delimitadores están equilibrados

Ingresa una expresión para validar

Ejemplos de uso:

({[]}) ✓ Equilibrado
([)] ✗ No equilibrado
(a + b) * [c - d] ✓ Equilibrado
{[()]} ✓ Equilibrado
((( ✗ No equilibrado
hola mundo ✓ Equilibrado

Información

Delimitadores válidos:

( ) [ ] { }

Reglas de validación:

  • Cada apertura debe tener su cierre correspondiente
  • Los delimitadores anidados deben cerrarse en orden inverso
  • Se ignoran letras, números y operadores
  • Una expresión sin delimitadores se considera equilibrada
